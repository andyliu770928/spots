# TRIPMARK DECK (Spots) 專案架構書 - 20260328 版

## 1. 本次調整
- 首頁卡片的分類顯示改為中文標籤，與上方 category filter 一致。
- 將原本面向使用者的 `status` 顯示改為 `rating`。
- 新增 `rating` 篩選列，支援 `沒去過` 與 `1-5 分` 條件。
- 景點內頁的「收藏狀態」改為「評分」。
- 基本資訊新增 `opening_hours`。

## 2. UI 規則
### 首頁卡片
- 顯示分類中文，例如 `美食`、`景點`
- 顯示評分標籤
  - 無評分：`沒去過`
  - 有評分：`N 分`
- 若有營業時間，顯示在卡片資訊區

### 首頁篩選
- 第一列：category filter
- 第二列：rating filter
  - `全部`
  - `沒去過`
  - `5 分`
  - `4 分以上`
  - `3 分以上`
  - `2 分以上`
  - `1 分以上`

### 內頁
- Hero 標籤顯示 `分類 + 評分`
- Quick Action 第三顆按鈕顯示 `評分`
- 基本資訊包含：
  - 地址
  - 評分
  - 營業時間
  - 備註 / 平台

## 3. 資料欄位
### places 新增
- `rating INT`
- `opening_hours TEXT`

### 相容策略
- 後端仍保留 `status`
- 若 `rating > 0` 且未指定 `status`，預設將 `status` 視為 `visited`
- UI 主要不再直接展示 `status`
